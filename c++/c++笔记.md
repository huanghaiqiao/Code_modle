c++汇总

```
一、c++ 在定义变量就是在向内存申请空间，申请空间后才有指针
```



#### 符号

```c++
char a = '';
std::string b ="";
// 字符用单引号
// 字符串用双引号
```

#### 指针

```C++
// 输出a的地址
int a=1;
std::cout << &a << "\n";

// 打印地址
int m = 1;
int *a=&m;
std::cout << a << "\n";  // 注意：当为cha指针时候必须加(void *)才能输出指针

// 定义指针时候赋初值
int * a=nullptr;
int * b=0;  // 部分c++不赋初值，指针会是随机数
```
